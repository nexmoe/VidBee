VidBee est un téléchargeur de bureau construit avec Electron et alimenté par yt-dlp. Il propose une interface claire et une gestion de file d’attente pour le téléchargement de vidéo et d’audio.

Cette documentation se concentre sur l’usage réel et les réglages, en particulier les téléchargements avec connexion et la configuration des cookies.

## Commencer ici

- [Protocole vidbee://](./protocol.mdx) : Téléchargement rapide via un protocole d’URL.
- [Cookies](./cookies.mdx) : Configurer les sessions connectées et le contenu restreint.
- [RSS](./rss.mdx) : S’abonner aux flux RSS et ajouter automatiquement les téléchargements.
- [FAQ](./faq.mdx) : Questions courantes et dépannage.
